---
title: 'Get Application'
description: 'Retrieve complete application details including customer and device information'
openapi: 'GET /applications/{application_id}'
---

<Note>
This endpoint requires API key authentication via the `x-api-key` header or `Authorization: Bearer <API key>`.
</Note>

## Example Request

```bash
curl -X GET \
  "https://your-project.supabase.co/functions/v1/applications/123" \
  -H "Authorization: Bearer leap_live_your_api_key_here" \
  -H "Content-Type: application/json"
```

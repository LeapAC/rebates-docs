## Setup

<Steps>
<Step title="Create your API Key">
  Generate API credentials from your partner dashboard for both sandbox and production environments.
</Step>

<Step title="Map devices to your system">
  Create a mapping between your product catalog and Leap's device IDs for eligibility checks.
</Step>

<Step title="Configure ownership of data requirements (Optional)">
  Define which data fields your system will provide vs. what customers will enter directly.
</Step>
</Steps>

## Get URL

[Get Incentives](/api-reference/endpoint/get-incentives) to retrieve a Connect URL for eligible customers in your App.

<Info>
  **Options:** API, CSV Upload
</Info>

### Request

```json
POST /incentives
{
    "address": string,
    "building_type": enum,
    "device_id": uuid,
    "reference_id": string,
    "create_application": true
}
```

## Place in Your App Flow

Place the offer to Claim Incentives inside your customer journey.

<CardGroup cols={3}>
<Card title="Post Purchase Email" icon="envelope">
  Send the incentive link in your order confirmation or follow-up emails
</Card>

<Card title="Order Confirmation" icon="circle-check">
  Display immediately after purchase completion
</Card>

<Card title="App Setup" icon="mobile">
  Integrate into your product setup or onboarding flow
</Card>
</CardGroup>

## Customer Fills the Application

Customer completes the application in Leap Connect. After completion, application_status is updated to `awaiting_partner` or `completed`.

<Note>
  The customer will provide required documentation and consent for data sharing during this step.
</Note>

## Upload Remaining Data Requirements

[Update Applications](/api-reference/endpoint/update-application) to provide information to complete the customer's application.

### Request

Use the [Update Application](/api-reference/endpoint/update-application) endpoint to upload partner data.

```json
PATCH /applications/{application_id}
{
  "customer": {
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "phone": "555-123-4567",
    "customer_devices": [
      {
        "id": 45,
        "purchase_price": 599.99,
        "installation_cost": 450.00,
        "purchase_date": "2025-01-05",
        "installation_date": "2025-01-10"
      }
    ]
  },
  "status": "customer_complete"
}
```
